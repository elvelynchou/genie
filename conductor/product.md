# Initial Concept
构建一个基于 Telegram 的“自进化”多智能体平台，利用 Gemini 的长上下文能力和 Redis Stack 的向量检索，实现复杂任务的自主拆解与执行。

---

# Product Guide: GenieBot (Bridge-03)

## Vision
To create an autonomous, self-evolving multi-agent Telegram orchestrator that intelligently manages complex tasks through a high-fidelity hybrid memory system and sophisticated reasoning loops.

## Core Goals
- **Autonomous Execution:** Seamlessly dismantle and execute complex user instructions via a multi-agent hierarchy.
- **Hybrid Memory (Graph-RAG):** Implement a hierarchical memory system (Strategy, Logic, Data) for precise context retrieval and long-term learning.
- **Stealth & Reliability:** Maintain high-stealth browsing and social media interaction via advanced automation engines (Camoufox/Nodriver).
- **Self-Evolution:** Enable the system to generate, sandbox, and deploy new agents dynamically based on observed needs.

## Key Features
- **Intelligent Orchestration:** Gemini-driven tool selection and multi-step reasoning.
- **Finance Pipeline:** Automated multi-source financial monitoring, structural cleaning, and associative archiving.
- **Social Publishing:** Automated X (Twitter) posting with session persistence and human-like interaction patterns.
- **Vision Integration:** OCR, prompt inversion, and multi-backend image generation (Vertex AI, Nanobanana).
- **Memory Consolidation:** Nightly "Dreaming Phase" to synthesize strategy nodes from raw factual data.

## Target Users
- Advanced users requiring personalized automation for research, finance monitoring, and social media management.
- Developers looking for a modular, self-expanding AI agent framework.
